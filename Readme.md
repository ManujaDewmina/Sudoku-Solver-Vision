# Run

1. For 9X9 Sudoku puzzles : `python sudoku_9.py` 
2. For 16X16 Sudoku puzzles : `python sudoku_16.py` 

Note: Sometimes, errors may occur in the number detection part (OCR) of a 16x16 Sudoku puzzle.
